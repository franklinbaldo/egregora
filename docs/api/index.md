# API Reference

The Egregora API is organized into several modules that correspond to the different stages of the processing pipeline. Each module provides specific functionality for handling different aspects of the personal knowledge graph creation process.

## Module Overview

- **Input Adapters**: Handle data ingestion from various sources
- **Privacy**: Implement PII detection and anonymization
- **Data Primitives**: Define core data structures used throughout the system
- **Enrichment**: Apply AI-powered analysis to your data
- **Transformations**: Pure functions for data transformation
- **Agents**: AI agents for content generation and processing
- **Database**: Data storage and retrieval layer
- **Output Adapters**: Format and export your knowledge graph
- **Orchestration**: Coordinate the processing pipeline
- **CLI**: Command-line interface components

Navigate to each module's documentation to see detailed API references for classes, functions, and protocols.
