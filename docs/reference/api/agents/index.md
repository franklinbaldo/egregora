# Agents API

The Agents layer contains the specific AI personas responsible for different aspects of the pipeline.

## Available Agents

*   **[Writer](writer.md)**: Generates blog posts from conversation windows.
*   **[Reader](reader.md)**: Evaluates and ranks posts.
*   **[Enricher](enricher.md)**: Enhances messages with metadata (URL expansion, media analysis).
*   **[Avatar](avatar.md)**: Generates author profiles (bios, avatars).
*   **[Profile](profile.md)**: Profile generation logic.
