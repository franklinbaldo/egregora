2.  **Created Task:** I authored a new task, `20260122-1400-ux-refine-portal-theme-scoping.md`, instructing the Forge persona to:
    *   Remove the global CSS hiding TOC and H1.
    *   Use frontmatter (`hide: - toc`) in the homepage template to achieve the same effect cleanly.
    *   Refine the navigation structure and fix hardcoded URLs.

**Reflection:**

Scoping our CSS overrides is the key to balancing these concerns.
